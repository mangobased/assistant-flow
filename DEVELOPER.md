# 🛠 Документация для разработчиков

## Архитектура продукта
Плагин разработан на **TypeScript** с использованием официального **Obsidian API**. 
Пайплайн обработки данных реализован в файле `main.ts`:
* Сбор данных: `this.app.vault.getMarkdownFiles()` с фильтрацией по `stat.mtime`.
* Анализ: Использование регулярных выражений (RegEx) для лексического парсинга задач и тегов.
* Визуализация: Интеграция синтаксиса `Mermaid.js` для нативного рендера графиков силами самого Obsidian.

## Инструкция по локальному запуску
Для сборки проекта из исходного кода потребуется Node.js.
1. Клонируйте репозиторий.
2. Выполните команду `npm install` для установки зависимостей.
3. Выполните `npm run build` для компиляции TypeScript в `main.js`.

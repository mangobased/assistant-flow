# 📊 Assistant Flow (Obsidian Plugin)

**Assistant Flow** — это интеллектуальный плагин для базы знаний Obsidian, который помогает осознанно отслеживать продуктивность, выявлять ключевые темы и динамику работы с задачами.

## 🎯 Что делает плагин (MVP v1.0)
1. Автоматически собирает измененные заметки за последние 7 дней.
2. Анализирует структуру: считает объем текста, находит открытые (`- [ ]`) и закрытые (`- [x]`) задачи.
3. Собирает частотность тегов (выявляет ключевые фокусы).
4. Генерирует наглядный Markdown-отчет с круговой диаграммой.

## 📦 Как установить
Поскольку плагин находится в стадии MVP, установка производится вручную:
1. Перейдите в раздел [Releases](../../releases) справа.
2. Скачайте архив `assistant-flow-v1.0.0.zip`.
3. Распакуйте файлы `main.js` и `manifest.json` в папку вашего хранилища: `.obsidian/plugins/assistant-flow/`.
4. В настройках Obsidian отключите "Безопасный режим" и включите плагин.

## 🚀 Как использовать
1. Откройте палитру команд Obsidian (`Ctrl+P` или `Cmd+P`).
2. Введите: **Создать аналитический отчет за неделю**.
3. Плагин автоматически создаст новую заметку с аналитикой и графиком.
